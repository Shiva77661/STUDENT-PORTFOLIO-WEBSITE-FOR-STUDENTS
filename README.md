# Student Portfolio Builder

A modern, colorful web application that allows students to create professional portfolios in minutes.

## Features

### 🎨 Beautiful Design
- Modern colorful UI with gradients
- Responsive design (mobile, tablet, desktop)
- Smooth animations and hover effects
- Card-based layout with glassmorphism effects

### 📝 Easy Portfolio Creation
- Simple form to enter all details
- Upload profile photo
- Add skills with proficiency levels
- Include education history
- Showcase projects with descriptions
- List achievements and certifications
- Add social media links (GitHub, LinkedIn, Email, Phone)

### 🎭 Multiple Templates
Choose from 3 professionally designed templates:
1. **Minimal Template** - Clean and professional
2. **Creative Colorful** - Vibrant and eye-catching
3. **Professional Dark** - Modern and sleek

### 🎨 Theme Customization
Change portfolio colors before downloading:
- Purple (default)
- Blue
- Pink
- Green
- Orange

### 📊 Portfolio Score Meter
Real-time feedback showing how complete your portfolio is based on filled fields.

### 📥 PDF Download
Export your portfolio as a professional PDF with one click.

### ✨ Live Preview
See your portfolio update in real-time as you make changes.

## How to Use

1. **Open `index.html`** in your browser to see the home page
2. **Click "Create Portfolio"** to start building
3. **Fill in your details** in the form (Step 1)
   - Watch the Portfolio Score Meter increase as you add more information
4. **Choose a template** (Step 2)
   - Select from Minimal, Creative, or Professional Dark
5. **Preview and customize** (Step 3)
   - Change theme colors
   - See live preview of your portfolio
6. **Download** (Step 4)
   - Click "Download as PDF" to save your portfolio

## Customization

You can easily customize:
- Colors in CSS variables (`:root` section in style.css)
- Add more templates by creating new template functions in script.js
- Add more theme colors in the `themes` object in script.js
- Modify form fields in builder.html

## Tips for Best Results

1. Fill in all sections for a complete portfolio (aim for 90%+ score)
2. Use a professional profile photo
3. Be specific in project descriptions
4. Include links to your work
5. Keep descriptions concise but informative
